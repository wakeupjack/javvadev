Conditionals (if/else, switch)

Pelajari struktur kontrol kondisional untuk membuat keputusan dalam kode.

If/Else Statement

if (score >= 90) {
  grade = "A";
} else if (score >= 80) {
  grade = "B";
} else {
  grade = "C";
}

Switch Statement

switch (day) {
  case "Monday":
    return "Start of week";
  case "Friday":
    return "TGIF!";
  default:
    return "Regular day";
}

Loops (for, while, forEach)

Pelajari berbagai jenis perulangan untuk memproses data secara efisien.

For Loop

for (let i = 0; i < 5; i++) {
  console.log(i);
}

While Loop

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

forEach

array.forEach(item => {
  console.log(item);
});

Asynchronous JavaScript

Pelajari cara menangani operasi asinkron dalam JavaScript.

Callbacks

function fetchData(callback) {
  setTimeout(() => {
    callback("Data loaded");
  }, 1000);
}

Promises

fetchData()
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error(error);
  });

Async/Await

async function loadData() {
  try {
    const data = await fetchData();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
